# s6

* https://github.com/just-containers/s6-overlay
* https://github.com/skarnet/s6

## 文件作用

`run`     服务如何启动

`down`    如果 `down` 文件存在，则 S6 不会启动该服务

`finish`  程序退出后的操作

`log/run` 启动日志服务
